About the Program:

Why I made it:

I developed this program after seeing how slow and manual the process of translating textbooks into braille could be. Translating braille by hand takes too long, and I wanted to create a faster solution so visually impaired students could access their learning materials in a timely manner. My program automates much of the braille translation process, supporting Arabic, Italian, and English to ensure that more students in diverse language communities can benefit. By speeding up the production of braille resources, this tool helps visually impaired students keep pace with their peers, breaking down barriers to education and providing a more inclusive learning experience across multiple languages.

How it works:

a. Entering Text Manually
a1. Enter Text: In the provided text box, type or paste the text you want to convert to braille.
a2. Convert Text: Click the Convert button. The conversion to braille will begin immediately.
a3. View Results: After conversion, you’ll be directed to the results page.

b. Uploading an Image
b1. Upload Image: Click the Upload Image button to select the image file you want to convert.
b2. Review Analyzed Text: After uploading, the program will analyze the text in the image, and you’ll be taken to a review page.
b2.1 Edit Text: Check the analyzed text for accuracy. Make any necessary corrections in the provided text box.
b3. Convert to Braille: When you’re satisfied with the text, click the Convert to Braille button.

c. Accessing and Using the Results
c1. Copy to Clipboard: On the results page, click the Copy button to save the braille text for easy access.
c2. Return to Homepage: Click the Home button to return to the main page and begin a new conversion if needed.

Frequently Asked Questions

Why is my Image Not being Converted?
There are several reasons why this issue may occur. First, ensure that the image is a screenshot of printed text or a scanned photo of printed text, as handwriting is not recognized by the program. Additionally, make sure the image is in PNG, JPG, or JPEG format. If you have confirmed these factors and the program still doesn't work, the image may be too large for processing. In this case, try inputting smaller sections of the image.

What should I do if the detected language is incorrect?
If the app misidentifies the language, you can try adjusting the image or using clearer text. Additionally, you can input the text manually if necessary.

Why is my Text Not being Converted?
There are 2 main reasons why this could be happening. Firstly, there is a chance that the text or input you used is not in the braille map mentioned below. Secondly, if you are using Arabic or Italian, the program expects at least a few words or a sentence for it to recognize the language, so try increase the amount of content.

How I made it:

The development of this program involved a combination of technologies including HTML, CSS, Flask, Python, Javascript, and Tesseract OCR to facilitate the conversion of text and images into braille. I began by designing a user-friendly interface using HTML and CSS, ensuring that the layout was both accessible and visually appealing for users. I then used Flask, a lightweight web framework for Python, which served as the backbone of the application, managing server-side logic and routing. I then implemented the API endpoints to handle user input and file uploads efficiently. Tesseract OCR API helped with allowing the extraction of text from images. This powerful tool helped prepare the analyzed visuals for conversion into braille format in all three languages.
